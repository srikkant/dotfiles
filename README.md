# srikkant's dotfiles

- **stow**: config management
- **aerospace**: window management (OSX)
- **zsh**: shell
- **ghostty**: terminal
- **tmux**: multiplexer
- **nvim**: editor
- **fzf**: fuzzy finder
- **antidote**: plugin manager for zsh (install to home)
- **zoxide**: jump to directories
